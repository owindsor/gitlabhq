To control permissons against a branch, the workflow is a little weird. First, you need to commit back to the repo so that you have atleast one branch Next, open the project from the project view by clicking on the project name

  • Click the 'Commits' tab
  • Then select the 'Branches' sub-tab
  • Select the 'Protected' sub-tab
  • Select that branch that you want to protect and click'Add'

The selected branch will now be only writable by users with a 'Master' permission on that project.
